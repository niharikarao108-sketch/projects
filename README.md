#  Healthcare Operations & Billing Analytics Dashboard

##  Project Overview

This project presents a Power BI dashboard analyzing hospital operations, patient demographics, billing trends, and insurance insights.  
It helps in improving decision-making using data-driven insights.

---

##  Objectives

- Analyze patient admissions and service utilization  
- Identify billing and cost trends  
- Study relationship between age and treatment cost  
- Evaluate hospital stay duration (LOS)  
- Analyze insurance provider patterns  

---

##  Dashboard Sections

### 1. Operations Dashboard
- Patient admissions & discharges  
- Length of Stay (LOS)  
- Monthly trends  

### 2. Financial Dashboard
- Revenue analysis  
- Treatment cost breakdown  

### 3. Quality Dashboard
- Readmissions  
- Infection rates  

### 4. Insurance Dashboard
- Insurance provider distribution  
- Claim patterns
